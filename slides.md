# Aufbau des Clusters
*Initialisieren der Knoten*

```bash {all|5,6|2-4|all} twoslash
cockroach start \ 
--insecure \ 
--advertise-addr= 10.0.2.15 \ 
--join= 10.0.2.15,10.0.2.14 \ 
--cache=.25 \
--max-sql-memory=.25 \ 
--background
```
